多智能体架构

PhishingDetector 由四个专业智能体组成流水线：语义意图分析、多维关联检测、风险研判、响应处置。每个智能体专注于特定任务，通过事件流向后端实时推送分析状态与结果。

语义意图分析 识别邮件意图、话术技巧与情感操控。
多维关联检测 评估发件人与 URL 的可信度，标记内容异常。
风险研判 综合评分并映射 ATT&CK 攻击技战术。
响应处置 生成隔离、告警或放行等处置建议。
🔌

API 接入

提供 /api/analyze/stream 流式接口，支持 SSE 实时返回分析事件，便于集成到现有安全运营平台。

🔐

数据安全

支持完全本地化部署，邮件内容仅在分析过程中使用，不会持久化存储或外传。

❓

常见问题

支持哪些邮件格式？

目前支持手动输入邮件主题、发件人和正文。未来将支持 .eml / .msg 文件上传与自动解析。

分析结果是否准确？

系统基于大语言模型与规则引擎结合，提供置信度与风险评分。建议对高风险邮件进行人工复核。

可以私有化部署吗？

可以。项目基于 Python FastAPI 构建，可替换为本地或私有 LLM，无需依赖外部服务。
